## Projeto de TCC
Aplicação web de videoaulas para ensino da língua inglesa.<br>
<b>Este projeto está sendo refeito.</b>

### Stack
`Ruby 2.7`<br>
`Rails 5.1`<br>
`Bootstrap 4`<br>
`SQLite`<br>
